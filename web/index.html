Flashing replaces the firmware on the connected ESP32-S3. Keep the device plugged in until the installer says it is finished.

Before You Click Install

  1. Use Chrome or Edge on macOS, Windows, Linux, or ChromeOS.
  2. Connect the device with a USB data cable.
  3. If the installer cannot connect, hold BOOT, tap reset or power-cycle, then try again.
  4. After flashing, place books on the SD card under /books.

The browser flasher is powered by ESP Web Tools and Web Serial. iOS Safari and Firefox do not currently support the required browser serial API.
